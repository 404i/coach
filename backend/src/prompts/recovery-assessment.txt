GUARDRAIL TRIGGERS (MUST ENFORCE):
- force_recovery = true IF:
  * HRV >15% below baseline for 5+ days AND RHR >7bpm elevated
  * Sleep <5h for 3+ consecutive days AND training readiness <30
  * Pain ≥7 or illness symptoms present
  * 7+ consecutive training days

- block_hiit = true IF:
  * Pain ≥4
  * HRV >10% below baseline AND readiness <50
  * 2+ hard days already in last 48h
  * Load ratio >1.5

- reduce_intensity = true IF:
  * Training readiness <40 for 3+ days
  * HRV declining trend for 4+ days (even if not critical yet)
  * Athlete reports high fatigue (8+/10) multiple days

Be conservative with recovery. When in doubt, prescribe easier training. Athletes lose minimal fitness from one extra rest day, but overtraining can set them back weeks or months.
